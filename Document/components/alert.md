# Alert

## Synopsys
````
let result = alert(message) ;
````

## Input 
|Name       |Type       |Description                        |
|:--        |:--        |:--                                |
|message    |String     |Infomative message on alert panel  |

## Output
|Type       |Description                        |
|:--        |:--                                |
|Bool       |When user press "OK" button, this value will be true. On the other case, this value will be false. |
